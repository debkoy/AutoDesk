BACHELOR OF SCIENCE COMPUTER ENGINEERING

(CADRAF)

AutoCAD is a powerful software program used for creating 2D and 3D designs. Developed by Autodesk in 1982, it was initially created for engineers and architects to leverage the growing power of personal computers for drafting tasks.

Capabilities

2D Drafting and Design: Create precise technical drawings with lines, shapes, dimensioning tools, and annotations.

3D Modeling: Build and visualize 3D models for objects, structures, and scenes.

Customization: Tailor the software to your workflow with industry-specific toolsets (architecture, mechanical engineering, etc.) and APIs for creating custom functions.

Collaboration: Share and edit designs with colleagues through cloud storage and online collaboration tools.

Command Meanings

Drawing Commands: These commands create basic geometric shapes like LINE, CIRCLE, ARC, RECTANGLE, etc.

Editing Commands: Modify existing geometry with commands like COPY, MOVE, TRIM, EXTEND, etc.

Annotation Commands: Add text, dimensions, and other explanatory elements to your drawings using commands like TEXT, DIM, LEADER, etc.

View Commands: Control how you view your drawing in the workspace with commands like ZOOM, PAN, ROTATE, etc.

Layer Commands: Organize your drawing elements into separate layers for better control using commands like LAYER, -ISOLATE, etc.
